# Chat Room (Capstone Project)

The **Chat Room** project is a dynamic web application used for real-time communication. It leverages the MERN stack along with `socket.io` for real-time messaging, allowing users to engage in instant conversations. (Build by Anant Manas)

## Technologies Used

- **MongoDB**: NoSQL database used for storing chat messages.
- **Express.js**: Node.js framework for building RESTful APIs.
- **React.js**: JavaScript library for building user interfaces.
- **Node.js**: JavaScript runtime environment for server-side logic.
- **socket.io**: Library for real-time, bidirectional and event-based communication.

## Features

- **Real-time Messaging**: Users can engage in real-time conversations.
- **Responsive Design**: User-friendly interface that adapts to different devices.
- **Scalability**: Built using the MERN stack to handle growing user base and messages.
